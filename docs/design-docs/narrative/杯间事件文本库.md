# 杯间事件文本库

> CS Cup Manager 两个杯赛之间的叙事事件
> 每个杯间产生 1~2 个事件，事件根据 placement（成绩）筛选

---

## 事件结构

```js
{
  id: "事件ID",
  severity: "mild" | "severe",     // 事件强度
  tone: "positive" | "negative" | "neutral",  // 基调
  placements: ["八强", "四强", ...], // 触发条件：杯赛成绩
  title: "显示标题",
  narrative: "长篇描述（当前阵营叙事）",
  passive: "被动影响描述（世界叙事，无选项）",
  options: [
    {
      id: "选项ID",
      label: "选项标签",
      result: "选择后的结果描述",
      delta: "属性变化摘要",
      effect: { firepower, discipline, tacticalExecution, cohesion }
    }
  ]
}
```
