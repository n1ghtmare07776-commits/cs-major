# 局内描述文本库

> CS Cup Manager 比赛中各阶段战斗描述模板
> 覆盖 11 种场景 × 5~6 条变体，总模板量 60+条

---

## 模板变量

所有文本支持以下变量替换：

| 变量 | 含义 |
|---|---|
| `{player_a}` `{player_b}` `{player_c}` `{player_d}` `{player_e}` | 己方 5 名选手 |
| `{opponent}` `{opponent_a}` `{opponent_b}` `{opponent_c}` `{opponent_d}` | 对手选手 |
| `{rival_team}` | 对手队名 |
| `{caller_name}` | 己方指挥 |
| `{star_player_name}` | 己方明星选手 |
| `{site}` | 地图点位（A点/B点/中路/香蕉道/小道等） |
| `{hp}` | 血量（随机 14-35） |
| `{time}` | 剩余时间（随机 10-25 秒） |
| `{weapon}` | 当前武器 |

## 1. opening — 开局交火（6条）

> 渲染第一波枪接触的紧张感

1. `{player_a} 第一身位突进 {site}，急停接两个点射——第二发修正到头。{opponent_a} 倒地时手里还捏着没交出去的闪光。`
2. `{rival_team} 先顶出来拿信息，{player_d} 的补闪在空中爆开。{opponent_b} 全白两秒，{player_a} 贴脸收掉首杀。`
3. `{player_c} 在 {site} 架了整整十几秒——准星没动，呼吸压着手。烟散开一条缝的瞬间，{opponent_a} 露了半个身位，枪声一秒就把首杀刻进记分板。`
4. `{player_c} 先开枪，第一发就接住了 {opponent_a} 的头线。{rival_team} 想补位，{player_a} 第二时间横拉过来，两人交叉火力直接钉死路口。`
5. `{site} 门口撞了个正着。{player_a} 和 {opponent_a} 同时看到对方，同一瞬间两人都开了枪。{player_a} 的头盔弹飞了，但 {opponent_a} 先倒了。`
6. `{player_d} 在 {site} 的烟边蹲了十秒，听到脚步声靠近才拉出去。不是反应快，是他一直在等这个声音。`

## 2. trade — 中期补枪交换（6条）

> 中盘阶段的人头互换，渲染拉扯感

1. `{rival_team} 回防没慢。{opponent_c} 从烟边补出来先带走 {player_a}，{player_b} 立刻横拉补枪。两边各掉一人，局面从 5v5 压成 4v4。`
2. `{player_d} 的烟把入口封住，{opponent_b} 混烟扫射先打残两人。{player_e} 没退，蹲在烟边把补枪接上，硬把人数换回同等。`
3. `{player_c} 先拿一枪，马上被 {opponent_d} 从侧翼换掉。这不是高光局，是补枪局——谁慢半秒，谁就少一个人。`
4. `{player_a} 和 {opponent_a} 在 {site} 门口互相点了两枪，然后各自被身后的补枪收掉。第一波交火打完，双方都没占到便宜，继续控默认。`
5. `{player_b} 横拉出去补枪时，{opponent_c} 已经从另一边探出来了。两个人同时开枪，同时倒地。场上瞬间从 4v4 变成 3v3。`
6. `{player_a} 被打残后躲在箱子后面，{player_d} 隔着烟穿了两枪把补位的 {opponent_b} 逼退。不是收人头，是保住残血的人能活到下一波。`

## 3. utility — 道具攻防（6条）

> 烟、火、闪的博弈，渲染策略感

1. `{player_d} 的燃烧弹砸在 {site} 入口，火墙把 {rival_team} 的回防卡了十几秒。对面不是不想打，是被道具按在门外等火灭。`
2. `{opponent_b} 的手雷落得很深，{player_a} 和 {player_e} 同时被炸成大残。血量没归零，但下一次对枪已经变成一发子弹的事。`
3. `双方在 {site} 门口连续交了烟、火、双闪。画面里全是白光和灰烟，真正的胜负不在准星，而在谁还能记住下一步该站哪。`
4. `{player_d} 的烟雾弹弧线拉得很高——落点刚好卡在 {opponent_b} 和包点之间。这一颗烟直接把对面阵型切成两半。`
5. `{opponent_b} 的火把 {player_a} 从掩体里逼出来，紧接着一颗闪在空中爆开。{player_a} 全白，只能凭记忆往后退。`
6. `双方道具交完的瞬间，画面突然干净了——没有烟，没有火，只有枪声。这时候才知道谁的道具留到了最后。`

## 4. highlight — 高光时刻（6条）

> 扫射转移、绕后、穿烟等个人高光

1. `{player_a} 连续接住三个身位。第一个人从左边出，第二个人从烟边补，第三个人想捡枪——全被同一个弹匣收掉。解说只喊了一个字。`
2. `{player_d} 消失了半分钟。等 {rival_team} 开始回防时，他已经绕到对方阵型后方，偷背身连拆两个补枪链。`
3. `{player_c} 在中路打出一枪穿烟。不是运气——他前两秒听到了脚步停顿，提前把准星放在烟边的常规身位。`
4. `{player_b} 在残局最后五秒极限拆包，{rival_team} 的交叉火线已经锁死了包点，但他拆包时刚好吃了一颗 {player_d} 续上的闪光掩护。`
5. `{player_a} 跳上箱子的一瞬间正好避开脚底手雷，落地就瞄着 {opponent_a} 的头。这种预判不是练出来的，是打了太多局之后长在肌肉里的。`
6. `{player_c} 的甩狙从左边拉到右边，准星没停过——先把前期压力顶住，再穿烟压住第二个身位，最后一个人也被他按在回防路上。观众席已经疯了。`
