# Core Beliefs

## Lightweight First

The game succeeds if a player can finish a season quickly and still feel that roster choices mattered.

## Text Is the Main Surface

Do not hide weak mechanics behind visuals. The match log and tactical prompts are the game.

## Realism Supports Fun

Use real teams and player names to make the world recognizable, but tune attributes for a better manager game.

## Determinism Enables Trust

Seeded simulation allows replays, tests, debugging, and balance conversations.
