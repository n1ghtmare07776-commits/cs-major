### 1.2 各经济等级的典型装备配置

> **写文本时的对照表**。检查你的事件中选手拿了什么武器——必须和本表一致。

#### 手枪局（Pistol Round）— 每人 $800

> ⚠️ **任何比赛的第 1 局都是手枪局。不存在例外。不存在"全买"。不存在"起步枪"。**

**T 方典型配置**：

| 配置 | 花费 | 装备 | 战术意义 |
|---|---|---|---|
| **全甲手枪** | $650 | Kevlar + Glock-18 | 最常见，拼枪硬 |
| **半甲+道具** | $500+$300 | Kevlar($650 的一半没有"半甲"选项) + 烟雾/闪光 | 备选，有道具配合 |
| **沙鹰豪赌** | $700 | Deagle 无甲 | 一枪头秒，但极脆 |
| **省钱** | $0 | Glock-18 裸奔 | 下局有钱 |

> **纠正**: CS 没有"半甲"($325)选项。$800 实际能买的组合：
> - 全甲($650) + Glock → 剩 $150（刚好一个闪光不够）
> - P250($300) + 闪光($200) + 烟雾($300) = $800（零甲，全道具流）
> - Tec-9($500) + 闪光($200) → 剩 $100
> - Deagle($700) → 剩 $100

**CT 方典型配置**：

| 配置 | 花费 | 装备 | 战术意义 |
|---|---|---|---|
| **全甲手枪** | $650 | Kevlar + USP-S/P2000 | 最常见 |
| **拆弹+道具** | $400+$400 | Kit + 烟雾($300) + 可选闪光 | 无甲，专注拆弹 |
| **P250 道具流** | $300+$200+$300 | P250 + 闪光 + 烟雾 = $800 | 零甲，精确打击 |
| **省钱** | $0 | USP-S 裸奔 | 下局有钱 |

#### ECO 局（纯经济局）— 每人 $0~1,200

> ECO 局的目标：**尽可能少花钱，活着保枪，或者赌一个奇迹击杀。**

| 配置 | 花费 | 装备 | 战术意义 |
|---|---|---|---|
| **纯保（Save）** | $0 | 默认手枪 | 最保守，最大化下局资金 |
| **P250 ECO** | $300 | P250 | 比默认手枪好一点 |
| **沙鹰 ECO** | $700 | Deagle 无甲 | 赌一枪头，高风险 |
| **五人凑道具 ECO** | ~$300/人 | 默认手枪 + 1-2 颗烟雾/闪光 | 全队凑钱买一颗烟雾掩护 rush |

> **禁止在 ECO 叙事中出现**: 步枪（AK/M4/Galil/FAMAS）、AWP、满道具、全甲+头盔。

#### 半买 / 弱起（Half-Buy / Light Buy）— 每人 $1,800~2,500

> 有钱但不够全买。典型：输了手枪局后的第 2 局 ECO 到第 3 局。

| 配置 | 花费参考 | 装备 |
|---|---|---|
| **冲锋枪+甲** | $1,050+$1,000 = $2,050 | MAC-10/MP9 + 全甲+头盔 |
| **Galil/FAMAS+甲** | $1,800+$1,000 = $2,800 | Galil($1,800) + 全甲+头盔 + 少量道具 |
| **Scout+甲** | $1,700+$1,000 = $2,700 | SSG 08 + 全甲+头盔，少量道具 |
| **沙鹰+全甲+道具** | $700+$1,000+$300 = $2,000 | Deagle + 甲+头盔 + 1 颗烟雾 |

#### 强起（Force Buy）— 每人 $2,700~3,500

> 把全部钱花光，赌赢这一把。通常是势均力敌但都不够全买的状态。

| 配置 | 花费参考 | 装备 |
|---|---|---|
| **Galil+全甲+少量道具** | $1,800+$1,000+$200 = $3,000 | T 方标配 |
| **FAMAS+全甲+少量道具** | $2,050+$1,000+$200 = $3,250 | CT 方标配 |
| **AK+头盔无甲?** | — | AK 无甲极不推荐，头部一枪就死 |
| **AWP+无甲** | $4,750 | 只够一把狙，没护甲非常危险 |

#### 全买（Full Buy）— 每人 $4,000+

> **除了手枪局以外，这是常态的最高装备水平。**

| 配置 | 花费参考 | 装备 |
|---|---|---|
| **AK 全买（T）** | $2,700+$1,000+$1,000 = ~$4,700 | AK-47 + 全甲+头盔 + 烟雾+2闪+燃烧+HE |
| **M4 全买（CT）** | $2,900+$1,000+$1,200 = ~$5,100 | M4A1-S + 全甲+头盔 + 烟雾+2闪+燃烧+HE + 拆弹套件 |
| **AWP 全买** | $4,750+$1,000+$600 = ~$6,350 | AWP + 全甲+头盔 + 烟雾+2闪 |

> **全买局的典型叙事画面**: 五个人满甲满道具，步枪手端着 AK/M4，狙击手扛着 AWP。每人至少 2-3 颗道具。烟雾弹封锁关键通道，闪光弹掩护进点。

#### Anti-ECO / Bonus Round — 特殊

> Anti-ECO：己方全买，对手 ECO。通常买冲锋枪（击杀奖励 $600）来"刷钱"。
> Bonus Round：不买新装备，保留上局存活时的装备。目标是"这把赢了白赚，输了还有钱下局全买"。

| 配置 | 花费参考 | 装备 |
|---|---|---|
| **Anti-ECO（T）** | ~$2,050 | MAC-10 + 全甲+头盔 + 道具 |
| **Anti-ECO（CT）** | ~$2,250 | MP9 + 全甲+头盔 + 道具 |
| **Bonus Round** | $0~少量 | 保留上局装备，至多补充道具 |

### 1.3 经济进度叙事速查

```
手枪局 $800/人
  ├─ 赢 → 第2局 ~$4,000+ → 可冲锋枪/步枪
  │   → 第3局 ~$5,000+ → 全买（可能 Anti-ECO 或 Bonus Round）
  └─ 输 → 第2局 ~$1,400+ → ECO（只有手枪，或最多沙鹰/冲锋枪）
      → 第3局 ~$3,300+ → 强起/半买（如果第2局 ECO 了）
      → 第4局 ~$5,000+ → 全买（连败奖励到顶 + ECO 省的钱）
```

### P1-1: `match-beats.ts` 全买 Beat 的"全员满钱"表述

**文件**: `src/content/match-beats.ts`  
**行号**: 1478–1482  
**事件 ID**: `fullbuy_setup`

**当前内容**（本身没有性错误，但需要加防护）:

```typescript

  "经济够了。全员满钱。\n\n" +
  "{player_a} 扛起了 AK-47，{player_b} 选了 M4A1-S，{player_c} 起了 AWP..."
```

**问题**: "全员满钱" 语义上接近 "$16,000 上限"——但实际全买只需要 ~$4,700-$6,350。如果写手不加区分地使用"满钱"，会给人一种"钱花不完"的错觉。

**修复方案**（微调措辞）:

```typescript
text:
  "经济够了。五个人都到了全买的门槛。\n\n" +
```

### P2-1: 统一 ECO 局叙事中的武器描述

**问题**: 当前多个文件中 ECO 局有时出现"手枪"、有时出现"沙鹰"、有时没有武器描述。

**规范**:

| 经济等级 | 叙事中允许的武器 | 禁止的武器 |
|---|---|---|
| 手枪局 | Glock-18, USP-S, P2000, P250, Deagle, Tec-9, Five-SeveN, Dual Berettas | **所有**步枪/冲锋枪/霰弹枪/狙击枪/AWP |
| ECO ($0-$1,200) | 默认手枪, P250, Deagle（仅限 $700 花光）, 偶尔一个烟雾 | 步枪, AWP, Scout（$1,700 贵了）, 冲锋枪 |
| 弱起 ($1,800-$2,500) | MAC-10, MP9, Galil, FAMAS, Scout, Deagle+甲 | AWP（除非不买甲）, AK/M4（除非不买甲+无道具） |
| 强起 ($2,700-$3,500) | Galil, FAMAS, Scout, 冲锋枪, **可以出现 1 把 AK/M4**（挤出来的） | AWP（$4,750 远超预算）, 满道具 |
| 全买 ($4,000+) | AK-47, M4A4, M4A1-S, AWP, AUG, SG553, **满道具** | — |

**检查方法**: 在事件文本中搜索武器名（AK-47、M4、AWP、Galil 等），确认上下文经济等级与上表一致。

### P2-2: 道具描述的具体化

**问题**: 当前道具描述通常是"满道具"或"烟雾闪光"这种笼统表达。

**改进方向**: 不同经济等级的道具详细程度不同。

| 经济等级 | 道具描述粒度 |
|---|---|
| 手枪局 | 提及具体的 1 颗道具（如"一颗烟雾封住 CT 架枪位"） |
| ECO | 道具稀少，如果出现了就是关键（"全队凑钱买了一颗烟雾"） |
| 弱起/强起 | 2-3 颗起，具体到类型（"两颗闪光一颗烟雾"） |
| 全买 | 可笼统写"满道具"但偶尔点出组合（"烟雾闪光燃烧弹齐飞"） |

**示例（好 vs 差）**:

```
❌ 差: "五个人满道具冲了出去。"
✅ 好: "三颗烟雾同时封住 A 大、A 小和连接——CT 瞬间什么都看不见了。两颗闪光跟上，五个人冲进 A 点。"
```

### P2-3: CS 没有"半甲"

**问题**: 设计文档和事件文本中多次出现"半甲"这个词。

**事实**: CS 的护甲只有两个级别：
1. **Kevlar Vest**（$650）：只有胸部防护
2. **Kevlar + Helmet**（$1,000）：头+胸完整防护

没有"$300 半甲"或"$500 半甲"的概念。"半买"（Half-Buy）指的是经济策略（花一半的钱），不是"买一半的甲"。

**需要修正的位置**:

| 文件 | 行 | 原文 | 改为 |
|---|---|---|---|
| `DECISION_OPTIONS_SPEC.md` | 77 | `$300 甲 + $200 闪光` | `P250($300) + 闪光($200)` — 删掉"$300 甲"这个不存在的概念 |
| `DECISION_OPTIONS_SPEC.md` | 89 | `半甲($300) + 拆弹套件($400)` | `P250($300) + 拆弹套件($400) → 剩 $100` — 同样修正 |
